1、JavaScript简介 1)JS是基于对象(提供了很多可以直接用的对象)和事件驱动(动态)的脚本语言，主要应用在客户端(浏览器) 2)特点 1.交互性:信息的动态交互 2.安全性:不能访问本地硬盘 3.跨平台性:能够支持JS的浏览器都可以运行 3)Java与JS的区别 1.Java是sun公司，现在oracle，JS是Netscape网景公司公司 2.Java是面向对象，JS是基于对象的， 3.Java是强类型的语言，JS是弱类型的语言 4.Java需要编辑，JS只需解析 4)JS由的三部分组成 1.ECMAScript:由ECMA(European Computer Manufacturers Association欧洲计算机制造商协会)制定的JS的语法 2.BOM:Broswer Object Model 3.DOM:Document Object Model 5)JS语法规范 1.变量、函数、运算符都区分大小写 2.建议在语句结尾添加";" 3.两种注释://和/**/ 2、JS与HTML的两种结合方式 1) 2) 由于html解析是从上到下，所以建议把script标签放到body后面 3、数据 1)JS有引用值(Heap中对象)和原始值(Stack中数据)两种数据类型 2)定义变量都使用var关键字 3)5种原始值数据类型:string、number、boolean、null、undifined，所有的引用类型都是Object 4)alert(typeof(null))返回object，alert(null == undefined)返回true 5)undefined是从值null派生来的，表示声明变量但未赋值，null表示对象不存在 6)typeof方法可以查看变量类型，instanceof方法用于判断一个变量是否某个对象的实例 4、运算符 1)JS不区分整数和小数:alert(123/1000*1000)返回123，但Java中返回0 2)加减运算问题: alert("456"+1)返回4561，alert("456"-1)返回455 alert("a"-1)返回NaN(表示不是一个数字) alert(true+1)返回2，alert(false+1)返回1 alert(true+"ab")返回trueab，alert(false+"ab")返回falseab 3)==比较的只是值，===比较的是值和类型 5、数组 1)var arr=[1,2,3];var arr=[1,"4",true];var arr = [];
